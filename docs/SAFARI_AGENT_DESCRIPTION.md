# Vision Safari 에이전트 (Safari Agent) 설명

이 문서는 'Vision Safari' 게임에서 동작하는 AI 에이전트인 '사파리 에이전트'에 대한 설명입니다.

## 1. 기본 목적 및 환경

*   **게임 플레이**: 'Vision Safari'라는 게임을 플레이하는 AI 에이전트입니다.
*   **목표**: 50x50 크기의 가상 사파리 환경에서 주어진 미션에 따라 특정 동물(타겟)을 찾아야 합니다.
*   **제한된 시야**: 에이전트는 전체 맵을 한 번에 볼 수 없으며, 자신의 현재 위치 주변 10x10 크기의 제한된 영역만 이미지 형태로 인식할 수 있습니다.
*   **환경 요소**: 사파리에는 에이전트 자신(P), 장애물인 나무(🌲), 그리고 찾아야 할 다양한 동물 이모지(🐯, 🐘 등)가 존재합니다.

## 2. 작동 방식 (Core Loop)

에이전트는 매 턴마다 다음과 같은 순서로 작업을 수행합니다:

1.  **관찰(Observe)**: 매 턴마다 현재 위치 주변의 10x10 시야를 이미지로 자동 제공받습니다.
2.  **추론(Reason)**: 제공된 이미지를 분석하여 주변에 타겟이 있는지, 장애물이 있는지 등을 파악하고 다음 행동을 계획합니다.
3.  **행동(Act)**: 계획에 따라 아래 설명된 특정 `도구(Tool)`를 사용하여 명령을 실행합니다.

## 3. 사용 가능한 도구 (Tools)

에이전트는 다음과 같은 4가지 주요 도구를 사용하여 게임을 진행합니다:

*   `move`: 플레이어를 상/하/좌/우로 이동시킵니다. 한 번에 최대 4개의 연속적인 이동(예: 오른쪽 3칸 → 위로 2칸)을 명령할 수 있으며, 중간에 장애물(나무)을 만나면 이동이 중단됩니다.
*   `update_notepad`: 제한된 시야를 극복하기 위한 '메모장' 기능입니다. 탐색한 지형, 장애물 위치, 동물을 목격한 좌표 등을 기록하여 맵을 스스로 그려나가는 데 사용됩니다. 이 메모는 미션이 끝나도 유지됩니다.
*   `declare_found`: 미션 목표인 특정 타겟을 찾았을 때(해당 동물 위로 이동했을 때) 호출하여 달성을 선언합니다.
*   `declare_done`: 주어진 미션의 모든 타겟을 찾았을 때 호출하여 임무가 완전히 종료되었음을 알립니다.

간단히 말해, **사파리 에이전트는 제한된 시야를 가진 AI가 `move`와 `update_notepad`를 활용해 스스로 맵을 탐험하고, 목표 동물을 찾아 `declare_found`와 `declare_done`으로 임무를 완수하는 지능형 에이전트**입니다.
